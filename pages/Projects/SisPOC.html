SisPOC

budget system for civil works - sistema presupuestario de obras civiles

(2021)

Venezuela, Miranda ,Chacao , Constructora Valkiria C.A.

Details:

SisPOC is a desktop application to elaborate Unit Price Analysis (a mathematical model which predicts the right cost that a product or service should have based on its specific properties or details) and budgets. saving the work time, manual calculations, reducing errors, comparing the materials and equipments qualities for determinate which one is the best according the requirements for each work and showing the prices in local and foreign currency (Like: Bolivar - Dolar).


Contains:

- Show prices in multi currency.

- Show Reports in pdf.

- Capture informations (materias - equipments - prices - currency value) from websites like BCV and MercadoLibre.

- Compairing materials - equipments qualities and prices.

- Percentage configurations.


Technologies:

C# - Entity Framework Core - SQL Server - HtmlAgilityPack.

Benefits:


Time optimization

Web Scraping

Systematized processes

Multi Currency
